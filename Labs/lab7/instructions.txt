Write a program that uses structure "Record" which has three fields:

ch - character type
 n - integer type
 x - double type

to declare an array, a, of size 10 of type Record, and uses function fill_array to populate all fields with:

ch - random small letters
 n - random integers in range [5, 10]
 x - random numbers in range [5, 10)

Use function print_array to print the contents of all fields of all Records in the array. 

EXAMPLE OUTPUT (estimated syntax, not full output):

i   ch  n   x
0   a   6   5.42
1   z   8   6.97
2   b   7   9.98
